CSS Rubric

Excellent Good Decent Bad
Code Comprehensibility Proper indents, comments are helpful Coding has a few issues but is otherwise understandable Code is understandable but only barely Code is not organized, does not function, and contains too many unused lines of code
Layout Aesthetically pleasing, organized, and contains all the essential elements Functional and understandable, but leaves room for improvement No style involved Layout gives you vertigo
Colors Good use of colors, at least 2 but no more than 4 that are pleasing to look at At least 1 additional color or more than 4 colors The colors are only slightly altered or are unpleasant to look at Too many very poorly matched colors that make the content unreadable, or no use of added colors
Functionality All the input areas work seamlessly Input has some difficulty but ultimately not too hard Some parts of the input works, but barely any Input generally does not work
