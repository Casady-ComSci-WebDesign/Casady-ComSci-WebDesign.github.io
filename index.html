Form Page Assignment


In this assignment, you will build the front-in to a form that could be used to collect profile information. Processing data requires using using server-side tools such as PHP. We have discussed various form elements, and how to style them. You should use this knowledge to

There are two parts to this assignment:

  1. Group Assignment: Build the rubric for the .css file and the .html file
  2. Individual Assignment: Make a form that takes the information that might be found in a profile when building a social media account. You will build the form only. Actually processing data is much more involved.
The groups have been assigned in class.


Goals:

Three of the goals for this assignment are:

  1. Work collaboratively using a team repository on GitHub
  2. Learn how to use GitHub to host a static page
  3. Create a style forms using HTML and CSS

Rubric Group Assignment:

You are to build the rubric for the HTML page that will contain the form or the CSS page that will style the page contianing the form. Remember that rubrics need to have some kind of specificity to them. Vague directions are difficult to follow, and vague requirements are frustrating to satisfy.
